Feature Scaling 特征缩放

不同特征的取值间相差较大时，可能会使梯度下降达到收敛过慢，这种情况下，重新缩放不同的特征，使它们都具有可比的取值范围，来speed
up Gradient Descent。

  1. 标准化：除以最大值 ==> 取值范围 0-1

  2. mean normalization(均值归一化): xi−μimax⁡xi−min⁡xi\frac{x_i - \mu_i}{\max \limits_{x_i} - \min\limits_{x_i}}xi​max​−xi​min​xi​−μi​​ 其中 μi\mu_iμi​为xix_ixi​的均值

  3. Z-score Normalization: xi−μiσi\frac{x_i - \mu_i}{\sigma_i}σi​xi​−μi​​, 其中μi\mu_iμi​为均值 σi\sigma_iσi​为标准差

Converge of Gradient Descent

  1. 学习曲线（learning curve）随迭代次数趋于某一水平渐近线

  2. 以ϵ\epsilonϵ作为成本函数J(w⃗,b)J(\vec{w},b)J(w,b)每次迭代减少的下限，当成本函数减少值<=ϵ<=\epsilon<=ϵ时，认为收敛

Identify problem with gradient descent

With a small enough α\alphaα ,J(w⃗,b)J(\vec{w},b)J(w,b) should **decrease **on every iteration

  1. 学习曲线上下波动 ==> 学习率α\alphaα过大
  2. 学习曲线不断增大 ==> code bug

Pick α\alphaα values technique

0.001 -> 0.003 -> 0.01 -> 0.03 -> 0.1 -> 0.3 -> 1 -> …

每次放大3倍或10倍

选取收敛时迭代次数最少的一个（或比之稍小的）α\alphaα值
